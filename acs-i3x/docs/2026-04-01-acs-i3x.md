# acs-i3x Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:executing-plans to implement this plan task-by-task.

**Goal:** Build a fully i3X-compliant read-only REST/SSE service for ACS that translates i3X queries into Factory+ service calls.

**Architecture:** New TypeScript ACS service using WebAPI (Express + Kerberos/Basic auth) from `js-service-api` and ServiceClient from `js-service-client`. Subscribes to `UNS/v1/#` for live values, queries ConfigDB for type system, Directory for device status, InfluxDB for history. In-memory object tree and value cache.

**Tech Stack:** TypeScript, Express (via WebAPI), ServiceClient, InfluxDB client, Jest, MQTT (via ServiceClient)

**Reference docs:**
- Design: `acs-i3x/docs/design.md`
- Pitch: `acs-i3x/docs/pitch.md`
- TID: `acs-i3x/docs/to-improve.md`
- i3X spec: https://github.com/cesmii/i3X (main branch)
- WebAPI source: `lib/js-service-api/lib/webapi.js`
- FplusHttpAuth source: `lib/js-service-api/lib/auth.js`
- Example service: `acs-example-service/`

```typescript
// ---- Response envelope ----

export interface I3xSuccess<T> {
    success: true;
    result: T;
}

export interface I3xError {
    success: false;
    error: { message: string };
}

export type I3xResponse<T> = I3xSuccess<T> | I3xError;

export interface I3xBulkItemSuccess<T> {
    success: true;
    elementId: string;
    result: T;
}

export interface I3xBulkItemError {
    success: false;
    elementId: string;
    error: { message: string };
}

export type I3xBulkItem<T> = I3xBulkItemSuccess<T> | I3xBulkItemError;

export interface I3xBulkResponse<T> {
    success: boolean;
    results: I3xBulkItem<T>[];
}

// ---- Data model ----

export interface I3xNamespace {
    uri: string;
    displayName: string;
}

export interface I3xObjectType {
    elementId: string;
    displayName: string;
    namespaceUri: string;
    sourceTypeId: string;
    schema: object;
}

export interface I3xObject {
    elementId: string;
    displayName: string;
    typeElementId: string;
    parentId: string | null;
    isComposition: boolean;
    isExtended: boolean;
}

export interface I3xObjectWithMetadata extends I3xObject {
    metadata: {
        description?: string;
        typeNamespaceUri: string;
        sourceTypeId: string;
        relationships?: Record<string, string | string[]>;
    };
}

export interface I3xRelationshipType {
    elementId: string;
    displayName: string;
    namespaceUri: string;
    relationshipId: string;
    reverseOf: string;
}

export type I3xQuality = "Good" | "GoodNoData" | "Bad" | "Uncertain";

export interface I3xVqt {
    value: unknown;
    quality: I3xQuality;
    timestamp: string;  // RFC 3339 UTC
}

export interface I3xValueResponse extends I3xVqt {
    elementId: string;
    isComposition: boolean;
    components?: Record<string, I3xVqt>;
}

export interface I3xHistoryItem {
    elementId: string;
    values: I3xVqt[];
}

// ---- Requests ----

export interface I3xBulkRequest {
    elementIds: string[];
}

export interface I3xValueRequest extends I3xBulkRequest {
    maxDepth?: number;  // default 1, 0 = infinite
}

export interface I3xHistoryRequest extends I3xBulkRequest {
    startTime: string;  // RFC 3339
    endTime: string;    // RFC 3339
    maxDepth?: number;
}

export interface I3xSubscriptionCreateRequest {
    clientId: string;
    displayName?: string;
}

export interface I3xSubscription {
    clientId: string;
    subscriptionId: string;
    displayName: string;
}

export interface I3xSubscriptionListRequest {
    clientId: string;
    subscriptionIds: string[];
}

export interface I3xSubscriptionDeleteRequest {
    clientId: string;
    subscriptionIds: string[];
}

export interface I3xRegisterRequest {
    clientId: string;
    subscriptionId: string;
    elementIds: string[];
    maxDepth?: number;
}

export interface I3xStreamRequest {
    clientId: string;
    subscriptionId: string;
}

export interface I3xSyncRequest {
    clientId: string;
    subscriptionId: string;
    lastSequenceNumber?: number;
}

export interface I3xSyncItem extends I3xVqt {
    sequenceNumber: number;
    elementId: string;
}

// ---- Server info ----

export interface I3xServerInfo {
    specVersion: string;
    serverName: string;
    serverVersion: string;
    capabilities: {
        query: { history: boolean };
        update: { current: boolean; history: boolean };
        subscribe: { stream: boolean };
    };
}
```

This is the most complex component. SSE streaming + sync polling + TTL cleanup.

**Step 1: Write failing tests**

Test cases:

Lifecycle:
- `create(clientId, displayName)` → returns `{ clientId, subscriptionId, displayName }`
- `create` with same clientId → creates separate subscription (multiple allowed)
- `list(clientId, subscriptionIds)` → returns matching subscriptions
- `delete(clientId, subscriptionIds)` → removes subscriptions and queued values
- `delete` unknown subscription → returns 404

Registration:
- `register(clientId, subId, elementIds, maxDepth)` → registers watch keys
- `register` idempotent — re-registering same elementId succeeds
- `unregister(clientId, subId, elementIds)` → removes watch keys

Sync:
- After register + value change → `sync(clientId, subId)` returns queued values with sequenceNumbers
- `sync` with lastSequenceNumber → acknowledges and removes older entries
- `sync` without lastSequenceNumber → returns all queued values (first call)
- `sync` when empty → returns empty array

Stream (SSE):
- `stream(clientId, subId, res)` → sets correct SSE headers
- `stream` → sends queued values immediately
- `stream` → sends new values as they arrive
- Only one stream per subscription (second attempt returns error)

TTL:
- Subscription not accessed within TTL → automatically deleted
- Accessing (sync/stream) resets TTL timer

**Step 3: Implement SubscriptionManager class**

```typescript
export class SubscriptionManager {
    private subscriptions: Map<string, Subscription>;
    private valueCache: ValueCache;
    private ttl: number;

    constructor(opts: { valueCache: ValueCache; ttl: number });

    create(clientId: string, displayName?: string): I3xSubscription;
    list(clientId: string, subscriptionIds: string[]): I3xSubscription[];
    delete(clientId: string, subscriptionIds: string[]): void;
    register(clientId: string, subscriptionId: string, elementIds: string[], maxDepth?: number): void;
    unregister(clientId: string, subscriptionId: string, elementIds: string[]): void;
    sync(clientId: string, subscriptionId: string, lastSequenceNumber?: number): I3xSyncItem[];
    stream(clientId: string, subscriptionId: string, res: Response): void;
}

interface Subscription {
    clientId: string;
    subscriptionId: string;
    displayName: string;
    registeredElements: Map<string, number>; // elementId → maxDepth
    queue: I3xSyncItem[];
    nextSequenceNumber: number;
    activeStream: Response | null;
    lastAccessed: number;
    ttlTimer: NodeJS.Timeout;
}
```

Key implementation details:
- `subscriptionId` generated with `crypto.randomUUID()`
- Listen to `valueCache.onValueChange()` — when a value changes, check all subscriptions for matching elementIds, queue the update
- SSE: set `Content-Type: text/event-stream`, `Cache-Control: no-cache`, `Connection: keep-alive`. Write `data: ${JSON.stringify([vqt])}\n\n` for each update. Flush queued values on connect.
- Sync: monotonic `sequenceNumber` per subscription, starting at 1. `lastSequenceNumber` acknowledges receipt.
- TTL: `setTimeout` per subscription, reset on any access. On expire, clean up and delete.

```typescript
import express from "express";
import { ObjectTree } from "./object-tree.js";
import { ValueCache } from "./value-cache.js";
import { History } from "./history.js";
import { SubscriptionManager } from "./subscriptions.js";
import { wrapResponse, wrapError, wrapBulkResponse } from "./mapping.js";
import { I3X_SPEC_VERSION, Version } from "./constants.js";
import type { I3xServerInfo } from "./types/i3x.js";

export class APIv1 {
    private objectTree: ObjectTree;
    private valueCache: ValueCache;
    private history: History;
    private subscriptions: SubscriptionManager;
    public routes: express.Router;

    constructor(opts: {
        objectTree: ObjectTree;
        valueCache: ValueCache;
        history: History;
        subscriptions: SubscriptionManager;
    });

    private setupRoutes(): express.Router {
        const api = express.Router();

        // i3X envelope middleware (wraps all responses)
        api.use(i3xEnvelope);

        // 503 readiness check
        api.use(this.readinessCheck.bind(this));

        // Explore
        api.get("/namespaces", this.getNamespaces.bind(this));
        api.get("/objecttypes", this.getObjectTypes.bind(this));
        api.get("/objecttypes/:elementId", this.getObjectType.bind(this));
        api.post("/objecttypes/query", this.queryObjectTypes.bind(this));
        api.get("/relationshiptypes", this.getRelationshipTypes.bind(this));
        api.get("/relationshiptypes/:elementId", this.getRelationshipType.bind(this));
        api.post("/relationshiptypes/query", this.queryRelationshipTypes.bind(this));
        api.get("/objects", this.getObjects.bind(this));
        api.get("/objects/:elementId", this.getObject.bind(this));
        api.post("/objects/list", this.listObjects.bind(this));
        api.get("/objects/:elementId/related", this.getRelated.bind(this));
        api.post("/objects/related", this.queryRelated.bind(this));

        // Query
        api.get("/objects/:elementId/value", this.getObjectValue.bind(this));
        api.post("/objects/value", this.queryValues.bind(this));
        api.get("/objects/:elementId/history", this.getObjectHistory.bind(this));
        api.post("/objects/history", this.queryHistory.bind(this));

        // Subscribe
        api.post("/subscriptions", this.createSubscription.bind(this));
        api.post("/subscriptions/list", this.listSubscriptions.bind(this));
        api.post("/subscriptions/delete", this.deleteSubscriptions.bind(this));
        api.post("/subscriptions/register", this.registerSubscription.bind(this));
        api.post("/subscriptions/unregister", this.unregisterSubscription.bind(this));
        api.post("/subscriptions/stream", this.streamSubscription.bind(this));
        api.post("/subscriptions/sync", this.syncSubscription.bind(this));

        // i3X error handler (returns error envelope)
        api.use(i3xErrorHandler);

        return api;
    }
}
```

Each handler is straightforward: validate input, call the relevant component, return data (the envelope middleware wraps it). Bulk endpoints iterate over elementIds and build per-item results.

```typescript
import { APIv1 } from "./api-v1.js";
import { I3X_SPEC_VERSION, Version } from "./constants.js";

import compression from "compression";

export function routes(opts: {
    objectTree: ObjectTree;
    valueCache: ValueCache;
    history: History;
    subscriptions: SubscriptionManager;
}) {
    const api_v1 = new APIv1(opts);

    return (app: any) => {
        // Gzip compression (MUST per i3X spec)
        app.use(compression());

        // /v1/info is OUTSIDE auth (mounted before WebAPI auth runs)
        // Actually — WebAPI sets up auth in init() before routes.
        // So we need to use the `public` option on FplusHttpAuth.
        // See auth.js line 41: if (opts.public) { this.public = pathToRegexp(opts.public); }
        // And line 65: if (this.public && this.public.test(req.path)) { req.auth = null; return next(); }
        
        // Info endpoint (lives on /v1/info, inside the v1 router but
        // allowed through auth via the `public` option on WebAPI)
        app.use("/v1", api_v1.routes);
    };
}
```

Note: The `public` option on FplusHttpAuth takes a path-to-regexp pattern. We pass `public: "/v1/info"` to WebAPI opts so that `/v1/info` is unauthenticated. The info endpoint handler lives inside api-v1 but checks `req.auth === null` (public access allowed).

**Step 2: Implement bin/api.ts**

Follow example-service pattern but with ServiceClient (not RxClient since we don't need Rx), plus MQTT and InfluxDB init.

```typescript
import { ServiceClient, UUIDs } from "@amrc-factoryplus/service-client";
import { WebAPI } from "@amrc-factoryplus/service-api";
import { routes } from "../src/routes.js";
import { ObjectTree } from "../src/object-tree.js";
import { ValueCache } from "../src/value-cache.js";
import { History } from "../src/history.js";
import { SubscriptionManager } from "../src/subscriptions.js";
import { Version } from "../src/constants.js";
import { GIT_VERSION } from "../src/git-version.js";
import pino from "pino";

let dotenv: any = null;
try { dotenv = await import("dotenv"); } catch (e) {}
dotenv?.config();

const { env } = process;
const logger = pino({ name: "acs-i3x", level: env.LOG_LEVEL || "info" });

// Init Factory+ service client
const fplus = await new ServiceClient({ env }).init();

// Build object tree from ConfigDB + Directory
const objectTree = await new ObjectTree({
    fplus,
    namespaceName: env.I3X_NAMESPACE_NAME!,
    namespaceUri: env.I3X_NAMESPACE_URI!,
    pollInterval: parseInt(env.I3X_POLL_INTERVAL || "60000"),
    logger,
}).init();

// Start value cache (subscribes to UNS/v1/#)
const valueCache = await new ValueCache({
    objectTree,
    staleThreshold: parseInt(env.I3X_STALE_THRESHOLD || "300000"),
    logger,
}).init(fplus);

// History module
const history = new History({
    influxUrl: env.INFLUX_URL!,
    influxToken: env.INFLUX_TOKEN!,
    influxOrg: env.INFLUX_ORG || "default",
    influxBucket: env.INFLUX_BUCKET || "default",
    objectTree,
});

// Subscription manager
const subscriptions = new SubscriptionManager({
    valueCache,
    ttl: parseInt(env.I3X_SUBSCRIPTION_TTL || "300000"),
});

// Start HTTP server
const api = await new WebAPI({
    ping: {
        version: Version,
        service: UUIDs.Service.Registry, // TODO: register an i3X service UUID
        device: env.DEVICE_UUID,
        software: {
            vendor: "AMRC",
            application: "acs-i3x",
            revision: GIT_VERSION,
        },
    },
    debug: fplus.debug,
    realm: env.REALM,
    hostname: env.HOSTNAME,
    keytab: env.SERVER_KEYTAB,
    http_port: env.PORT || 8080,
    public: "/v1/info",  // Unauthenticated per i3X spec
    routes: routes({ objectTree, valueCache, history, subscriptions }),
}).init();

logger.info("Starting acs-i3x server");
api.run();

// Re-poll timer
setInterval(() => objectTree.refresh(), parseInt(env.I3X_POLL_INTERVAL || "60000"));
```
