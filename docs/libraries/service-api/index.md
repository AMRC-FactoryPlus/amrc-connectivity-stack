# NodeJS Factory+ Web Service API

This is a NodeJS module providing the basic framework of an HTTP API
complying with the Factory+ Service spec.

This framework is used to develop the **server** side of ACS service.

It consists of 3 main components: 
- WebAPI [read](webapi.md)
- FplusHttpAuth [read](fplushttpauth.md)
- Notify [read](notify-v2.md)
